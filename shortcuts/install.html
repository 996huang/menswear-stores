📸 安装门店上传快捷指令

选择你的门店，一键安装快捷指令

📱 安装说明：
  1. 用 iPhone 的 Safari 浏览器 打开本页面
  2. 点击你门店旁边的 「安装」 按钮
  3. 系统会弹出「快捷指令」安装确认 → 点击「添加快捷指令」
  4. 安装后，桌面会出现快捷指令图标，点击即打开上传文件夹

💡 手动添加快捷指令的方法：
  1. 打开 iPhone 的 「快捷指令」 App
  2. 点击右上角 + → 添加操作
  3. 搜索 「文件」 → 选择 「打开文件夹」
  4. 将文件夹路径设置为你的门店上传文件夹
  5. 点击顶部名称，重命名为 「📸 [门店名]-上传产品图」
  6. 点击右上角 「添加到主屏幕」

💡 也可以用 「文件」App → 找到门店文件夹 → 长按 → 「添加到个人收藏」 实现快速访问
